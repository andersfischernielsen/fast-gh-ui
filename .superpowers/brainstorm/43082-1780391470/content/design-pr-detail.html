Conversation Tab

  • PR description rendered at top (markdown → HTML)
  • Threaded comments below, each with author avatar, timestamp, markdown body
  • Reply input at bottom — write comment, posts via Octokit

Commits Tab

  • Flat list of commits with author, message, SHA, timestamp
  • Click to expand — shows commit diff in the diff viewer
  • Not paginated initially (most PRs have <30 commits)

Checks Tab

  • List of check suites/runs from GitHub's Checks API
  • Each check: name, status (queued/in_progress/completed), conclusion (success/failure/neutral/skipped)
  • Color-coded badges (green/red/yellow)
  • Click to expand logs/details

Inline Comments on Diffs

  • Each diff line has a + icon on hover (left gutter)
  • Click opens an inline textarea below that line
  • Type comment → POST via Octokit pulls.createReviewComment
  • Existing inline comments shown below their line with thread
  • Comments attached to position in diff (line number + side), not absolute line — survives rebase
  • Pending review draft tracked in local state, submitted as batch
